# PRE PRUEBA 2

Material de preparación para la segunda prueba de **Programación Orientada a Objetos**.

## Contenido

| Archivo | Descripción |
| --- | --- |
| [Kimbo's Street Fights.pdf](<Kimbo's Street Fights.pdf>) | Enunciado de preparación. |
| [peleadores.txt](<peleadores.txt>) | Archivo de entrada para el problema. |

## Conceptos esperados

- Modelado de dominio a partir de un enunciado.
- Identificación de clases, atributos y responsabilidades.
- Lectura de archivos.
- Relaciones entre objetos.
- Separación entre carga de datos, lógica del sistema y presentación.
- Preparación para ejercicios de mayor tamaño propios de Prueba 2.

## Estado

Esta carpeta contiene el enunciado y los datos de entrada. No incorpora una solución Java completa dentro del repositorio.

## Uso recomendado

Primero intenta diseñar el modelo sin programar: clases, atributos, relaciones y métodos principales. Después define cómo cargar `peleadores.txt` y qué operaciones debería ofrecer el sistema.
